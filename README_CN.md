## 功能特色

**数学运算练习**
- 加法、减法、乘法、除法
- 分数运算（同分母/异分母、混合运算）
- 每种运算 6 个递进难度等级

**游戏化学习**
- 答对题目和完成关卡获得金币
- 收集贴纸和成就徽章
- 每日挑战，连续打卡奖励
- 商店系统：解锁主题、头像边框、装饰

**儿童友好设计**
- 选择角色：数学小王子或数学小公主
- 精美的粉色/蓝色主题世界
- 成就达成时的彩带庆祝效果
- 温暖鼓励的反馈语句

**学习工具**
- 错题回顾和针对练习
- 进度统计和报告
- 游戏时间限制，家长 PIN 保护
- 自定义难度范围

**更多特性**
- 支持离线使用 (PWA)
- 深色模式
- 多语言支持（中、英、西、法、德）
- 多用户档案
- 数据本地存储，保护隐私

## 诞生故事

> 这个应用诞生于陪女儿做数学练习的时光。我在想："如果有一个有趣的、充满魔法的应用，能让学数学变成一场冒险，那该多好啊！" 于是，数学王国就这样诞生了！

如果你有让它变得更好的想法，欢迎在 GitHub 上告诉我！祝所有小数学家们学业进步！
